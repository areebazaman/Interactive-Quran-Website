Step-by-Step Setup
1. Clone the Repository

git clone https://github.com/areebazaman/Interactive-Quran-Website.git

cd Interactive-Quran-Website

2. Install Backend Dependencies

cd backend
npm install

3. Install Frontend Dependencies

cd ../frontend
npm install


4. Launch the Application

Terminal 1 - Start Backend Server:

cd backend
node server.js

Terminal 2 - Start Frontend Development Server:

cd frontend
npm start

5. Access the Application
Navigate to http://localhost:3000 in your browser to begin using the Interactive Quran Recitation Web App with real-time pronunciation feedback.
